# Portfolio website

Single-page software engineer portfolio (HTML, CSS, vanilla JavaScript).

## Structure

- `index.html` — page structure and content placeholders
- `css/styles.css` — base styles (expanded in Phase 2)
- `js/main.js` — interactivity (Phase 3+)
- `assets/` — images, icons, `resume.pdf`

Open `index.html` in a browser or serve the folder with any static file server.
